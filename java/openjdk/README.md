## 概要
Java(openjdk)の最新バージョンでRemote Containersを起動する。
バージョン: openjdk 15.0.2 (2021/3/10現在)
また、起動時に、拡張機能 Java Extension Packのインストールを行う。

### 拡張機能
- Java Extension Pack
    - Language Support for Java
    - Debugger for Java
    - Java Test Runner
    - Maven for Java
    - Project Manager for Java
    - Visual Studio IntelliCode
